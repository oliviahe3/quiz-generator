
  # Study Guide Upload Website

  Allows users to upload 1 or more .txt, .md, or .pdf files containing lecture slides/notes/materials. Then uses a Gemini API key to generate personalized practice problems based on specific course content. 
  
  Integrates a seemless user interface allowing students to practice multiple choice and short answer questions while receiving instant feedback. 

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
  
  Run `npm start` to start the backend server.
